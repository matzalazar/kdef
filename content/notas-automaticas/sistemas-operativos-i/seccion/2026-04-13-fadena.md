# Resumen de la Unidad 1 - Semana 2 - FADENA

## Conceptos clave
- **Hardware**: Componente físico de un sistema informático, incluye computadoras y periféricos.
- **Software**: Parte lógica del sistema informático, compuesta por programas que indican a la computadora qué hacer.
- **Dato**: Elemento aislado que representa una medición o estado, no procesado.
- **Información**: Conjunto de datos procesados y relacionados que tienen significado.
- **Informática**: Conjunto de conocimientos y técnicas para el tratamiento automático de la información mediante computadoras.
- **Computación**: Procesos e instrucciones que realiza la computadora, relacionados con cálculos y operaciones lógicas.
- **Modelo de Von Neumann**: Arquitectura de computadoras que describe la estructura y funcionamiento de un sistema informático.
- **Diseño conceptual**: Modelo abstracto de cómo funcionará la computadora.
- **Unidad central de proceso (CPU)**: Componente que controla la operación de la computadora y ejecuta instrucciones.
- **Periféricos**: Dispositivos externos que se conectan a una CPU para comunicar la computadora con el mundo exterior.

## Desarrollo principal
Las computadoras son máquinas programables que procesan datos mediante instrucciones, permitiendo realizar tareas de manera rápida y eficaz. La diferencia entre datos e información es crucial: los datos son la materia prima, mientras que la información es el producto elaborado que puede ser interpretado. La informática automatiza actividades a través de computadoras, facilitando el tratamiento de grandes volúmenes de datos.

La arquitectura de computadoras implica crear un modelo que define la organización y funcionamiento de los componentes. La arquitectura de Von Neumann es fundamental para la mayoría de las computadoras modernas, compuesta por CPU, memoria y módulos de entrada/salida. La CPU es el único componente que puede procesar instrucciones, y su rendimiento depende de varios factores, incluyendo la frecuencia del reloj interno. Los buses son esenciales para la comunicación entre componentes.

Los periféricos son vitales para la funcionalidad de la computadora, ya que sin ellos no se pueden recibir órdenes ni proporcionar información. Se clasifican en periféricos de entrada, salida y almacenamiento. La placa base interconecta todos los elementos de la computadora, y el chipset coordina el intercambio de información.

El procesador ejecuta instrucciones secuencialmente, incrementando el contador de programa después de cada lectura. El sistema operativo gestiona el hardware y la interfaz de usuario, mientras que el software se clasifica en base a su función: software de base, software de aplicación y software para programación.
